\subsubsection*{template$<$class Socket\+Type$>$\newline
class Horizon\+::\+Networking\+::\+Socket$<$ Socket\+Type $>$}
\doxylink{class_a}{A} \doxylink{class_horizon_1_1_networking_1_1_socket}{Socket} object that handles a single connection. Sockets are moved into the thread by \doxylink{class_horizon_1_1_networking_1_1_socket_mgr}{Socket\+Mgr}, once accepted or connected. Once started, the object blocks to handle I/O events and requires explicit stopping. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Socket\+Type} & The type of the \doxylink{class_horizon_1_1_networking_1_1_socket}{Socket} object. The \doxylink{class_horizon_1_1_networking_1_1_socket}{Socket} object is templated to allow the \doxylink{class_horizon_1_1_networking_1_1_socket_mgr}{Socket\+Mgr} to handle different types of \doxylink{class_horizon_1_1_networking_1_1_socket}{Socket} objects. The \doxylink{class_horizon_1_1_networking_1_1_socket}{Socket} object must inherit from the \doxylink{class_horizon_1_1_networking_1_1_socket}{Socket} class. The \doxylink{class_horizon_1_1_networking_1_1_socket_mgr}{Socket\+Mgr} will call the \doxylink{class_horizon_1_1_networking_1_1_socket_a0771eb7f7860874879f379cf383f561e}{start()} method of the \doxylink{class_horizon_1_1_networking_1_1_socket}{Socket} object. The \doxylink{class_horizon_1_1_networking_1_1_socket}{Socket} object must implement the \doxylink{class_horizon_1_1_networking_1_1_socket_a0771eb7f7860874879f379cf383f561e}{start()} method. The \doxylink{class_horizon_1_1_networking_1_1_socket_a0771eb7f7860874879f379cf383f561e}{start()} method is called once from the \doxylink{class_horizon_1_1_networking_1_1_network_thread}{Network\+Thread}. The \doxylink{class_horizon_1_1_networking_1_1_socket}{Socket} object must implement the \doxylink{class_horizon_1_1_networking_1_1_socket_afb9cddbb3529c0f90977aa394a0fe6e9}{update()} method. The \doxylink{class_horizon_1_1_networking_1_1_socket_afb9cddbb3529c0f90977aa394a0fe6e9}{update()} method is called every n nanoseconds from the \doxylink{class_horizon_1_1_networking_1_1_network_thread}{Network\+Thread}. \\
