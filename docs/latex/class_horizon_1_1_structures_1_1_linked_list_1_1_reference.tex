\subsubsection*{template$<$class TO, class FROM$>$\newline
class Horizon\+::\+Structures\+::\+Linked\+List\+::\+Reference$<$ TO, FROM $>$}
This class is used to manage a linked list of References. It is not intended to be used directly. It is used as a base class for other classes that need to manage a linked list. The \doxylink{class_horizon_1_1_structures_1_1_linked_list_1_1_head}{Head} class is a friend of this class (derived from \doxylink{class_horizon_1_1_structures_1_1_linked_list_1_1_element}{Element}) to allow it to access the private members of this class. The relationship between the \doxylink{class_horizon_1_1_structures_1_1_linked_list_1_1_reference}{Reference} and the Reference\+Manager is a one-\/to-\/many relationship, where the Reference\+Manager is the one and the \doxylink{class_horizon_1_1_structures_1_1_linked_list_1_1_reference}{Reference} is the many. The Reference\+Manager is responsible for creating and destroying the \doxylink{class_horizon_1_1_structures_1_1_linked_list_1_1_reference}{Reference}. The Reference\+Manager is also responsible for calling the \doxylink{class_horizon_1_1_structures_1_1_linked_list_1_1_reference_a0093058835ce6a1d75c9f0743b1bd979}{link()} and \doxylink{class_horizon_1_1_structures_1_1_linked_list_1_1_reference_ab30fcbfaabb1c5209c72f9870467d80d}{remove()} methods of the \doxylink{class_horizon_1_1_structures_1_1_linked_list_1_1_reference}{Reference}. This creates a dynamic responsibility chain between the Reference\+Manager and the \doxylink{class_horizon_1_1_structures_1_1_linked_list_1_1_reference}{Reference}.
