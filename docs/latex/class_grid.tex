class Grid$<$ GRID\+\_\+\+OBJECT\+\_\+\+TYPES $>$}
\doxylink{class_grid}{Grid} is used to store objects in a 2\doxylink{class_d}{D} grid. This is used to speed up the search for objects in a given area. The grid is divided into cells of a given size. Each grid-\/cell contains a list of objects that are in that grid-\/cell. The grid is actually stored in the \doxylink{class_grid_holder}{Grid\+Holder} class.
\begin{DoxyItemize}
\item The \doxylink{class_grid_holder}{Grid\+Holder} class is a container for Grids.
\item The \doxylink{class_grid}{Grid} class is a container for objects.
\item The \doxylink{class_grid_reference_container}{Grid\+Reference\+Container} class is a container for objects in a grid-\/cell.
\item The \doxylink{class_grid_reference_container_visitor}{Grid\+Reference\+Container\+Visitor} class is used to visit the objects in a grid-\/cell.
