\doxysubsection{Detailed Description}
The \doxylink{class_task_scheduler}{Task\+Scheduler} class provides the ability to schedule std\+::function\textquotesingle{}s in the near future. Use \doxylink{class_task_scheduler_a35c0891c7aded13c6ae7ecd2e6d480a9}{Task\+Scheduler\+::\+Update} to update the scheduler. Popular methods are\+:
\begin{DoxyItemize}
\item Schedule (Schedules a std\+::function which will be executed in the near future).
\item Schedules an asynchronous function which will be executed at the next update tick.
\item Cancel, Delay \& Reschedule (Methods to manipulate already scheduled tasks). Tasks are organized in groups (uint), multiple tasks can have the same group id, you can provide a group or not, but keep in mind that you can only manipulate specific tasks through its group id! Tasks callbacks use the function signature void(\+Task\+Context) where \doxylink{class_task_context}{Task\+Context} provides access to the function schedule plan which makes it possible to repeat the task with the same duration or a new one. It also provides access to the repeat counter which is useful for task that repeat itself often but behave different every time (spoken event HLogs for example). 
\end{DoxyItemize}
